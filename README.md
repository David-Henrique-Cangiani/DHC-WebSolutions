# DHC WebSolutions

Site e plataforma de apresentação da **DHC WebSolutions** — meu negócio de criação de **sites institucionais e landing pages de alta conversão**, com foco em performance, SEO técnico e geração de leads.

🔗 **No ar:** [dhc-web-solutions.vercel.app](https://dhc-web-solutions.vercel.app/)

## Sobre o projeto

Da estratégia ao deploy: estrutura completa de uma oferta de serviço, com seções de serviços, processo, portfólio de projetos, planos e contato. Pensado para converter visitantes em clientes.

## Destaques

- Design moderno e responsivo (mobile-first)
- Foco em performance e tempo de carregamento
- Estrutura preparada para SEO
- Portfólio de projetos reais e seção de planos
- Integração de contato (WhatsApp / formulário)

## Tecnologias

- HTML5, CSS3 e JavaScript
- Deploy contínuo na Vercel

## Como rodar localmente

```bash
git clone https://github.com/David-Henrique-Cangiani/DHC-WebSolutions.git
cd DHC-WebSolutions

# Abra o index.html no navegador, ou rode um servidor local:
npx serve .
```
